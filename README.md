# Amalgam-Reffect-Pack
A more visible UI element for Engineer's Amalgam Elite Specialization

Tracks Evolve and Strain durations. 

<img width="424" height="281" alt="image" src="https://github.com/user-attachments/assets/07943e30-81b1-42ac-b303-dc799873ce4f" />

Also tracks Willing Host.

<img width="378" height="232" alt="image" src="https://github.com/user-attachments/assets/94ce33b4-af74-49ce-a798-73ecd739bfea" />


If Silver Lining is equipped, strains will be tracked individually. (They will display on top of Morph skills, although their position will not necessarily match the corresponding Morph skill, as reffect doesn't allow to track both buff and skill cooldowns at the same time).

<img width="392" height="222" alt="image" src="https://github.com/user-attachments/assets/87f81386-386b-4b4d-8121-0444e4148d6f" />


# How to use
First, download the zip file to your reffect root folder like so:

<img width="608" height="228" alt="image" src="https://github.com/user-attachments/assets/2e290a86-58b5-4d8d-b220-0c635f62b06c" />

Then right click > Extract here (the necessary files will be added to the packs and icons subfolders).

Then, return to reffect ingame and refresh packs.

Finally, enable / disable whichever interface size you prefer:
